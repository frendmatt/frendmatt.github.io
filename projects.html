Projects

Graph Bandwidth Minitization

Created a java command line program which took a graph data structure in the
form of an edge list. It then used my best attempt heuristic that used backtracking
to solve for the minimum bandwidth. It would then return the smallest graph
bandwidth it could find as well as how long it took to complete the computation.

Check it out on github!

Recursive DNS Resolver

Created a python command line program that uses DNSPython.
It will output to a text file returning all the calls it made and the final answer.
The output file also contains information such as query time, and the size of the message.
The github project contains data that I collected from the resolver compared to other resolvers.

Check it out on github!
